### Dev 파트

Dev 파트는 개발에 관련된 Tool 사용법을 정리하는 저장소입니다.
1. editor :: 개발 IDE에 관련된 문서입니다.
2. git    :: Git 사용법에 관한 문서입니다.
3. oxs    :: Mac 운영체제의 사용법에 관한 문서입니다.
4. remote :: 원격 컨트롤에 관련한 문서입니다.
