### 01. Tween 객체

```js
const TWEEN_OBJECT = createjs.Tween.get(container).to().call()
```

다음과 같이 얻은 TWEEN_OBJECT는

- pause()
- play()
- gotoAndPlay()

명령어를 이용해서 진행중에 중지 및 실행 시킬 수있습니다.

> play() 명령어를 사용해주면 loop 명령어를 그대로 가져갈 수 있습니다.
