#### 01. table

```html
    <table>
        <tr>
            <th></th>
            <th></th>
        </tr>
        <tr>
            <td></td>
            <td></td>
        </tr>
    </table>
```

table 태그는 표를 만드는 태그 입니다.
간단한 표를 만들때는 유용하나 성능상의 문제로 점점 사용하지 않고 있는 추세입니다.
그러나 간단한 프로토타입을 만들때는 유용합니다.

#### 02. table 가운데 정렬

table 태그는 absolute 영역으로 잡혀있는 듯 합니다.
그래서 text-align으로 가운데 정렬을 할 수 없습니다.

```
margin : 0px auto;
```
위와 같은 코드를 사용하면 가운데 정렬을 할 수 있습니다.
